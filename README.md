# Keystone Stack

**From Concept to Capital** — A deal intelligence platform for real estate developers, municipal governments, economic development organizations, investors, and CDFIs.

## What It Does

Keystone Stack helps users structure, evaluate, and package real estate development deals with a focus on incentive programs and capital stacking.

- **Deal Screener** — Quick go/no-go evaluation for restaurant, retail, office, multifamily, and mixed-use deals with instant health scoring
- **Deal Memo Generator** — 5-step wizard that produces a professional, lender-ready one-page deal summary with PDF export
- **7 Financial Simulators** — Pro formas, capital stacks, sources & uses, scenario comparison, and more
- **17 Incentive Programs** — Federal, state (PA), and local programs with eligibility details and actionable checklists
- **AI Business Coach** — Built-in knowledge base covering all incentive programs, financial concepts, and Erie market data
- **Glossary** — Development finance terminology explained in plain language

## Tech

Single-file React 18 application. No build step, no dependencies to install. Just open `index.html` in a browser.

- React 18 + Babel standalone (in-browser JSX compilation)
- jsPDF for branded PDF exports
- SheetJS for Excel exports
- SVG-based charts and visualizations
- CSS custom properties for theming

## Quick Start

```
git clone https://github.com/YOUR_USERNAME/keystone-stack.git
cd keystone-stack
open index.html
```
